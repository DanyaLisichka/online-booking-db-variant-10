# Вопрос 10

Золотухин Даниил гр. 454
---

Объясните, как ваш третий запрос (с оконной функцией или подзапросом) решает конкретную бизнес-задачу. Напишите краткий сценарий использования.
---

Третий запрос в  базе данных использует подзапрос для поиска клиентов, которые обращались за консультацией чаще среднего количества посещений. Этот запрос решает бизнес-задачу анализа активности клиентов и помогает психологическому центру выявлять наиболее постоянных посетителей.

Пример:

SELECT DISTINCT c.full_name
FROM clients c
JOIN appointments a
    ON c.client_id = a.client_id
GROUP BY c.client_id
HAVING COUNT(a.appointment_id) >
(
    SELECT AVG(cnt)
    FROM
    (
        SELECT COUNT(*) AS cnt
        FROM appointments
        GROUP BY client_id
    ) AS stats
);

Внутренний подзапрос сначала подсчитывает количество консультаций для каждого клиента, затем вычисляется среднее количество посещений среди всех клиентов. Основной запрос сравнивает количество записей каждого клиента со средним значением и выводит только тех клиентов, у которых число консультаций выше среднего.

Бизнес-сценарий использования данного запроса заключается в анализе клиентской активности. Например, администрация психологического центра может использовать результаты для формирования программы лояльности, предоставления скидок постоянным клиентам или оценки загруженности специалистов. Также запрос помогает определить клиентов, которые регулярно посещают консультации и потенциально нуждаются в длительном сопровождении специалиста.
