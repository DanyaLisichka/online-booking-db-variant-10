## Описание предметной области

Разработана база данных для системы онлайн-записи в психологический центр. Система позволяет клиентам регистрироваться, записываться на очные и онлайн-консультации, оставлять заявки и взаимодействовать с психологами. Для специалистов реализована возможность работы с различными психологическими методиками, а также ведение внутренних заметок по консультациям.

В проекте предусмотрены ограничения целостности данных, внешние ключи, триггеры и SQL-запросы для анализа данных. Дополнительно реализована проверка, запрещающая клиенту записываться к одному психологу чаще одного раза в три дня.

## Структура репозитория

В корне репозитория расположен файл `README.md` с описанием проекта. SQL-скрипт создания базы данных и заполнения тестовыми данными находится в файле `schema.sql`. Файл `queries.sql` содержит SQL-запросы для выборки данных, аналитики и проверки ограничений. ER-диаграмма базы данных хранится в каталоге `docs/er-diagram`. PDF-отчёт по проекту расположен в папке `docs/report`. Дополнительные материалы и скриншоты выполнения запросов находятся в директории `docs/screenshots`.
